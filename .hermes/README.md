# `.hermes/` Workspace Support

This folder contains templates and notes for running the existing Enterprise AI Workflow with Hermes Agent.

Nothing in this folder should contain real secrets. Runtime secrets belong in `~/.hermes/.env`.

## Files

- `config.template.yaml` — starter Hermes config/profile values.
- `env.example` — local environment variable template.
- `gateway/slack.md` — Slack gateway setup notes.
- `mcp/*.md` — MCP server setup notes.
- `skills/README.md` — skill migration notes.
